Push() / Pop() and translate()

Push begins its own drawing group that contains its own styles and transformations and pop ends the drawing group. Within the following example I also experiment with translate(), a function that shifts everything that comes after according to new x and y coordinates.

Random() and noLoop()

In class we created a looping function which fills the canvas with 50x50px squares. In addition to this, we introduced the ‘random’ attribute with the let ‘chosenColour’. This enabled us to input a range of colours which would be chosen at random to ‘fill’ each square. The noLoop() attribute is added to the setup function so that the act of choosing a random colour only happens once.

Yayoi Kusama - Inspired art:

Using the techniques discussed so far; and, I think, partially inspired by all the circles, I created an algorithmic piece that responds to Yayoi Kusama’s artwork. The work uses a nested ‘while’ loop that fills the canvas with circles. However, within the nested loop there is a push/pop, inside which is a translate() function with its coordinates set to random. The effect of this is that random creates a small offset variation to the new drawing reference points introduced with translate, this creates a more organic structure where the circles aren't perfectly spaced.

Final Outcome

My research for this project in addition to this experimentation has led me to want to focus more on recreating analog art with p5.js.
